Crochet Companion


Link To Code

Overview:
• Full-stack web application for crochet pattern search, enabling user authentication and robust search filters for a personalized experience

• Engineered REST API server with Next.js framework to enable user accounts and secure user-generated uploads

• Fun Fact - Crocheting is my favorite hobby! This project was so fun for me to make :)


Skills:
Netlify Identity · Next.js · React.js · HTML · JavaScript · CSS


Key Features:
• Create an account/login.


• Search for a pattern based on various attributes: Type, Difficulty, Colors, or Stitch Type.


• Upload your own pattern by typing in the values.
